Example code for pyGrunn presentation
=====================================


Run demo
========

.. code:: bash

   $ git clone https://github.com/lewicki/pygrunn_robot_demo.git
   $ cd pygrunn_robot_demo
   $ virtualenv .
   $ . bin/activate
   $ python bootstrap.py
   $ bin/buildout
   $ bin/sphinx-build -b html docs build
